Welcome to CA Application Performance Management community

Available extensions:

1. Apache CXF Tracer - This java agent extension allows tracing of Apache CXF web services.

2. EJB Intelligent Naming - A solution to solve EJB metric explosion and EJB naming problem.

3. JBoss Fuse

Other build packs & examples:

1. CA APM Command Center API Scripts

2. CA APM Example for Tomcat in Docker

3. CA Service Desk

4. IIB10 Extension - This extension helps monitor IBM Integration Bus v10 previously known as WebSphere Message Broker.

5. Nginx - Monitors nginx status and reports metrics to the CA APM EPAgent REST interface.

6. RabbitMQ

7. RedHat Enterprise Linux - This is a series of epagent plugins for monitoring both the OS and application processes (specifically WebSphere Application Server).

8. Simple Network and Server Health Check - Monitor servers and ports by PING and SOCKET connections

9. Solaris

10. CloudFoundry buildpack for .NET Core - This is a CloudFoundry buildpack for .NET Core, HWC and binary buildpack.
