Welcome to CA Application Performance Management Community

This page replaces the CA APM Marketplace. Some of the extensions that were available on the marketplace are included in the latest release of the product. You can find other available CA APM extensions here or on https://github.com/CA-APM. If you need help open an issue on https://github.com/CA-APM or email team-apm-marketplace.pdl@broadcom.com.

Learn how to contribute to the CA APM Developer Community here.

Available extensions:

  1. Apache CXF Tracer - This java agent extension allows tracing of Apache CXF web services.
  2. EJB Intelligent Naming - A solution to solve EJB metric explosion and EJB naming problem.
  3. JBoss Fuse
    4. - Java agent extension for JBoss Fuse. Key focus areas are the Apache Camel framework, Web Service coverage and JMS implementations.
  4. IBM WebSphere Portal
    5. - Java agent extension for IBM WebSphere Portal v8.
  5. IBM WebSphere DataPower
    6. - The CA APM Monitor for IBM WebSphere DataPower agent provides essential IBM WebSphere DataPower appliance performance data.
  6. IIB10 Extension - This extension helps monitor IBM Integration Bus v10 previously known as WebSphere Message Broker.

Other build packs & examples:

  1. CA APM Command Center API Scripts
  2. CA APM Example for Tomcat in Docker
  3. CA Service Desk
  4. Nginx - Monitors nginx status and reports metrics to the CA APM EPAgent REST interface.
  5. RabbitMQ
  6. RedHat Enterprise Linux - This is a series of epagent plugins for monitoring both the OS and application processes (specifically WebSphere Application Server).
  7. Simple Network and Server Health Check - Monitor servers and ports by PING and SOCKET connections
  8. Solaris - server montoring for Solaris
  9. CloudFoundry buildpack for .NET Core - This is a CloudFoundry buildpack for .NET Core, HWC and binary buildpack.
